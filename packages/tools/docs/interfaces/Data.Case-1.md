# Interface: Case

[Data](../modules/Data.md).Case

`Case` represents a datatype similar to a case class in Scala. Namely, a
datatype created using `Case` will, by default, provide an implementation
for a constructor, `Hash`, and `Equal`.

**`Since`**

2.0.0
